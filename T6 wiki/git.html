Instalacion git

Para instalar git en el servidor necesitaremos ayuda de unos cuantos comandos

  1. Para la instalación ejecutaremos
    “sudo apt-get install git”
  2. Después tendremos que vincularlo con un correo ya creado en git
    “git config --global user.email “T6euskera@retoProxmox.com”
  3. Iniciamos y creamos un repositorio en el directorio de trabajo
    “git init”
  4. Hacemos la primera copia:
    “git add”
  5. Y consolidamos:
    “git commit -m “consolidación de prueba”
  6. Para ver el registro de consolidaciones:
    “git log”
  7. Para ver el estado de git por si tenemos que hacer un “add” o una consolidación nueva:
    “git status”

Repositorio github

Nos damos de alta en GitHub.com

  1. Vinculamos a nuestro git (local) el nuevo directorio de trabajo (repositorio de GitHub)
    “git remote add origin HTTPS:\\------------------”
  2. Pasamos el proyecto de Git a GitHub:
    “git push -u origin master”
  3. Para recuperar un proyecto de GitHub:
    “git clone HTTPS:\\--------------------”
  4. Para pasar a GitHub modificaciones:
    “git push”
  5. Para traerte las modificaciones de otros compañeros(una vez hecho el clone):
    “git pull”
