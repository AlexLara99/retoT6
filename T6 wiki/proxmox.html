Qué es Proxmox?

Proxmox Virtual Environment es una solución completa de virtualización, que funciona sobre Debian GNU/Linux, permitiendo la creación y gestión de máquinas virtuales OpenVZ y KVM.

Proxmox no crea ningún sistema de virtualización (o paravirtualización) nuevo, sino que usa dos ya existentes (KVM y OpenVZ), y ofrece una interfaz común de gestión, que simplifica el trabajo y hace que no sea necesario conocer el interior de estos sistemas para poder trabajar con máquinas virtuales.

Está basado en Software Libre, es decir, que podemos ver y hacer lo que queramos con el código fuente. Proxmox es uno de los productos más completos para virtualización en entornos de servidor, capaz de hacerle sombra a opciones de pago como VMware.

Cómo instalar Proxmox

Para comenzar con la instalación de Proxmox, primero tendremos que crear un medio booteable, en este caso nosotros hemos utilizado un USB. Para poder crear el USB booteable utilizaremos un programa llamado Etcher.

Preparacion

Primero vamos a descargar la ISO de Proxmox desde la web oficial de Proxmox (https://www.proxmox.com/en/downloads/category/iso-images-pve) y también descargaremos Etcher desde su web oficial(https://etcher.io/). Una vez descargados, vamos a crear el USB, para ello abrimos el Etcher y seleccionamos la ISO, una vez seleccionada la ISO seleccionaremos el USB y le daremos a “Flash!”.Haz click aqui

Instalacion

Para comenzar con la instalación debemos introducir el USB en el servidor, una vez introducido en el servidor el USB, tenemos que iniciar el servidor desde el USB. Una vez iniciado el servidor desde el USB vamos a comenzar con la instalación de Proxmox. Tendremos que aceptar los términos y condiciones de uso del software. Una vez aceptado los términos y condiciones, una vez aceptadas nos solicitara que seleccionemos el disco duro donde queramos que se instale.

Después nos pedirá que seleccionemos nuestra localización y zona horaria, tras rellenarlo damos a siguiente y nos pedirá que introduzcamos una contraseña para poder administrar el servidor y un correo electrónico donde recibiremos notificaciones por parte del servidor. Ahora nos pedirá que introduzcamos los datos de dirección IP, mascara de red y gateway para el servidor, y finalmente comenzara la instalación del servidor Proxmox en el servidor. Haz click aqui
